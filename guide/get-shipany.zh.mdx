## 新用户操作流程

之前未购买过 “ShipAny 高级版” 的用户，可以按照以下步骤获取 ShipAny 高级版

<Steps>
<Step>购买 ShipAny 高级版</Step>

访问 [ShipAny 定价页面](https://shipany.ai/zh/pricing) -> 点击 "获取 ShipAny" 按钮 -> 下单并完成支付

支付完成后，自动购买“ShipAny Template Two”模板，并跳转到已购买模板页面。

<Step>查看已购买模板</Step>

在 [个人中心 -> 活动 -> 模板页面](https://shipany.ai/zh/activity/templates) 可以查看你的所有已购买模板。

<Step>激活模板</Step>

选择一个已购买模板，点击 "激活" 按钮 -> 在激活页面输入你的 GitHub 用户名 -> 提交激活

<Step>访问代码仓库</Step>

在已购买模板列表页面，选择一个模板，点击已激活的 “Github 用户名”，访问模板对应的代码仓库。

<Step>开始使用</Step>

参考 [快速开始](/zh/docs/quick-start) 文档，开始使用 ShipAny。

</Steps>

## 高级版用户获取新模板

之前购买过“ShipAny 高级版” 的用户，可以按照以下步骤获取新模板

<Steps>
<Step>查看购买记录</Step>

在 [个人中心 -> 设置 -> 支付页面](https://shipany.ai/zh/settings/payments) 可以查看你的所有购买记录。

正常情况下，这里会有一条“ShipAny 高级版”的支付订单。

<Step>购买新模板</Step>

在 [ShipAny 模板市场](https://shipany.ai/zh/templates) 可以查看所有已上架的模板，你可以按需购买。

“ShipAny 高级版”用户可以使用专享价购买模板市场中的模板，包括官方发布的模板和第三方开发者发布的模板。

> ShipAny 高级版用户可以 0 元购买 ShipAny 官方发布的两套基础模板

1. [ShipAny Template One](https://shipany.ai/zh/templates/shipany-template-one)
2. [ShipAny Template Two](https://shipany.ai/zh/templates/shipany-template-two)

<Step>查看已购买模板</Step>

在 [个人中心 -> 活动 -> 模板页面](https://shipany.ai/zh/activity/templates) 可以查看你的所有已购买模板。

<Step>激活模板</Step>

选择一个已购买模板，点击 "激活" 按钮 -> 在激活页面输入你的 GitHub 用户名 -> 提交激活

<Step>访问代码仓库</Step>

在已购买模板列表页面，选择一个模板，点击已激活的 “Github 用户名”，访问模板对应的代码仓库。

<Step>开始使用</Step>

参考 [快速开始](/zh/docs/quick-start) 文档，开始使用 ShipAny。

</Steps>

## 非高级版用户升级高级版

之前购买了“ShipAny 入门版” / “ShipAny 标准版”的用户，可以通过 微信 / [邮箱](mailto:support@shipany.ai) / [Discord](https://discord.com/invite/x9usYqTaaj) 等渠道联系作者，补差价升级“ShipAny 高级版”。

也可以在不升级的情况下，进行以下操作。

<Steps>
<Step>查看购买记录</Step>

在 [个人中心 -> 设置 -> 支付页面](https://shipany.ai/zh/settings/payments) 可以查看你的所有购买记录。

正常情况下，这里会有一条“ShipAny 入门版” 或 “ShipAny 标准版”的支付订单。

<Step>查看已购买模板</Step>

在 [个人中心 -> 活动 -> 模板页面](https://shipany.ai/zh/activity/templates) 可以查看你的所有已购买模板。

正常情况下，这里会有一条“ShipAny Template One” 的购买记录。

<Step>购买新模板</Step>

在 [ShipAny 模板市场](https://shipany.ai/zh/templates) 可以查看所有已上架的模板，你可以按需购买。

非高级版用户需要使用模板标价购买模板，不享受优惠价格。

</Steps>

## FAQ

- 之前购买过，找不到订单了，怎么解决？

1. 确认你在 [ShipAny 官网](https://shipany.ai/zh) 登录的邮箱，跟你购买时的邮箱一致。

2. 分别尝试一下谷歌登录和 Github 登录。

3. 确认在 [个人中心 -> 设置 -> 支付页面](https://shipany.ai/zh/settings/payments) 看不到订单。

通过 微信 / [邮箱](mailto:support@shipany.ai) / [Discord](https://discord.com/invite/x9usYqTaaj) 等渠道联系作者，提供购买时的邮箱和 Github 用户名，修复订单。

- 之前获得的免费版本，可以获取新版本吗？

免费赠送的版本，可以直接访问 ShipAny Template One 高级版代码仓库。在 ShipAny 官网没有订单记录，不能享受 ShipAny 高级版权益。

如需获取 ShipAny Template Two，或其他业务模板，需要在官网购买 ShipAny 高级版。

- 有什么优惠活动吗？

1. 双十一半价活动，持续到 2025.11.30，可以在 [ShipAny 官网](https://shipany.ai/zh/pricing) 半价购买 ShipAny 高级版。

2. “哥飞的朋友们”社群成员，可以使用专属折扣码，半价购买 ShipAny 高级版，长期有效。

3. “[1024 全栈开发](https://1024.is)”社群成员，可以使用专属折扣码，半价购买 ShipAny 高级版，长期有效。社群续费赠送 ShipAny 高级版。

- 无编程基础，可以购买吗？

建议懂点 HTML / CSS / JavaScript 基础，熟悉 Typescript / React / NextJS 更佳。

完全零基础需要有一定的学习能力，至少你能借助 AI 搞定本地开发环境搭建，能够通过 AI 辅助编程，实现自定义功能。

ShipAny 模板降低了开发门槛，但是有点编程基础可以更好的使用。
