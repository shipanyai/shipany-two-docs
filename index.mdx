## What is ShipAny?

ShipAny is a NextJS-based AI SaaS development framework with built-in rich components and business functions to help you quickly publish your own products.

The current document is for the use of ShipAny Two. For the use of ShipAny One, please refer to [ShipAny One Documentation](https://docs.shipany.ai).

## ShipAny Two

ShipAny Two is the second version of ShipAny, with extremely rich functions, including the following parts:

1. Core System

- Landing Page
- User Center
- Admin Dashboard

2. Core Modules (core)

- Database (drizzle-orm)
- Authentication (better-auth)
- Access Control (rbac)
- Internationalization (next-intl)
- Multi-theme Switching (themes)
- Documentation System (fumadocs)
- Credit System (credits)

3. Extension Modules (extensions)

- Payment (payment)

Support stripe / creem / paypal

- Storage (storage)

Support cloudflare r2 / aws s3

- Email (email)

Support resend

- AI Generation (ai)

Support openrouter / replicate / fal / kie

- Analytics (analytics)

Support google analytics / clarity / plausible / openpanel / vercel analytics

- Advertising (advertising)

Support adsense

- Affiliate Marketing (affiliate)

Support affonso / promotekit

- Customer Service (customer-service)

Support tawk / crisp
