## 什么是 ShipAny?

ShipAny 是一个基于 NextJS 的 AI SaaS 开发框架，内置丰富的组件和业务功能，帮助你快速发布自己的产品。

当前文档针对的是 ShipAny Two 的使用说明，ShipAny One 的使用说明请参考 [ShipAny One 文档](https://docs.shipany.ai)。

## ShipAny Two

ShipAny Two 是 ShipAny 的第二个版本，功能极其丰富，包括以下几部分：

1. 核心系统

- 着陆页（Landing Page）
- 用户中心（User Center）
- 管理后台（Admin Dashboard）

2. 核心模块（core）

- 数据库（drizzle-orm）
- 登录鉴权（better-auth）
- 权限控制（rbac）
- 国际化（next-intl）
- 多主题切换（themes）
- 文档系统（fumadocs）
- 积分系统（credits）

3. 扩展模块（extensions）

- 支付（payment）

支持 stripe / creem / paypal

- 存储（storage）

支持 cloudflare r2 / aws s3

- 邮件（email）

支持 resend

- AI 生成（ai）

支持 openrouter / replicate / fal / kie

- 数据统计（analytics）

支持 google analytics / clarity / plausible / openpanel / vercel analytics

- 广告（advertising）

支持 adsense

- 联盟营销（affiliate）

支持 affonso / promotekit

- 客服（customer-service）

支持 tawk / crisp
